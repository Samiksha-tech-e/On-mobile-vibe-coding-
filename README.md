# Mobile Hand Gesture Tracking System via Google Colab

An AI-powered, real-time hand gesture tracking and recognition software built using the modern **MediaPipe Tasks API** and **OpenCV**. This project is uniquely optimized to run entirely on a **mobile browser** using Google Colab, leveraging cloud infrastructure to bypass mobile hardware limitations.

## 🚀 Features
* **Zero Laptop Required:** Developed and executable entirely on a smartphone.
* **Modern MediaPipe Tasks API:** Utilizes the updated 2026 landmark framework for hand tracking.
* **Live Video Streaming Bridge:** Custom JavaScript implementation to stream native mobile camera feeds into a cloud hosted Jupyter notebook environment.
* **Real-time Gesture Analysis:** Automatically un-mirrors the video feed and detects distinct postures including an **Open Palm**, a **Fist**, or a **Single Pointing Finger**.

## 🛠️ How to Run on Your Phone
1. Click the `Untitled3.ipynb` notebook file inside this repository.
2. Tap the **Open in Colab** badge at the top of the file.
3. Switch your mobile browser settings to **"Desktop Site"** mode for easier navigation.
4. Go to the top menu, tap **Runtime**, and select **Run All**.
5. When prompted by your browser, grant **Camera Permissions** to the page.
6. Hold your front-facing camera up and test the tracking!
